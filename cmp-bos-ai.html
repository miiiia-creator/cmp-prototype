Description
Detects accounts that engage in pre-arranged profit-taking trades: buy-side and sell-side orders show mirrored execution within a short interval, with sell price significantly higher than buy price, implying coordinated counterparty arrangement.
Rule Info
Parameters
A/B Test
Change Log
Feature Expression
SELECT account, symbol, orderDate,
    COUNT(orderId) AS orderCount,
    SUM(CASE WHEN action='BUY' THEN tradeQty ELSE 0 END) AS buyTradeCount,
    SUM(CASE WHEN action='SELL' THEN tradeQty ELSE 0 END) AS sellTradeCount
FROM trade_events
WHERE isProfit = TRUE
  AND priceDiffRatio >= :priceDiffRatio
  AND orderTradeInterval <= :orderTradeIntervalMinute
GROUP BY account, symbol, orderDate
HAVING buyTradeCount >= :orderTimes AND sellTradeCount >= :orderTimes
